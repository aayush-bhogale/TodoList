# ToDo List Application (ASP.NET Core MVC)

## Overview

A simple ToDo List application built with ASP.NET Core MVC as part of my learning journey in .NET development.

The application allows users to create, view, edit, and delete tasks. It uses SQL Server with Entity Framework Core for data storage and demonstrates the fundamentals of ASP.NET Core MVC.

## Features

* Add new tasks
* View all tasks
* Edit existing tasks
* Delete tasks
* Store data in SQL Server using Entity Framework Core

## Technologies Used

* ASP.NET Core MVC
* C#
* Razor Views
* Entity Framework Core
* SQL Server
* Dependency Injection

## What I Learned

Through this project, I gained hands-on experience with:

* MVC Architecture (Models, Views, Controllers)
* CRUD Operations
* Routing and Model Binding
* Dependency Injection
* Entity Framework Core
* SQL Server Integration
* Code-First Migrations
* Repository Pattern

This project helped me understand how a real-world ASP.NET Core MVC application is structured and how data can be managed using Entity Framework Core and SQL Server.
